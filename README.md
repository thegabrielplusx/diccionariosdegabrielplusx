El Generador de Diccionarios (Gabrielplusx) fue diseñado para crear diccionarios personalizados destinados a pruebas de penetración y auditorías de contraseñas.El programa permite exportar los resultados en formato .txt y descargar la información OSINT que hayas ingresado durante el proceso.🕵️‍♂️

Ventajas:
Es de código abierto, usa la licencia GPL 3.0 y todo se ejecuta de manera universal en tu navegador. 

😎Idiomas disponibles: español, inglés, portugués de Brasil, portugués de Portugal.

Link de la página para probar el proyecto:
https://elgeneradordediccionariosgabrielplusx.netlify.app/

index.html (Esqueleto HTML)
diseño.css (estilos de la página CSS)
